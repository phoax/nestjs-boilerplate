# NestJS Boilerplate

/!\ Still under development. No unit tests and auth features are not finished.

## Description
This boilerplate breaks with NestJS modules mechanism.s It organizes code by folders instead of by features.

## Features
NestJS Boilterplate features:
- Login user
- Signup user
- Reset user password
- Command line
- Database orm with postgres
- Guards

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Commands

```bash
# get commands help
$ npm run command:dev -- --help
```

Seed
```bash
# get commands help
$ npm run command:dev seed
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
